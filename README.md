Customer Shopping Behavior Analysis

An end-to-end Data Analytics project using Python for data cleaning, PostgreSQL for querying and Power BI for visualization.
This project analyzes customer shopping behavior to uncover meaningful patterns and insights that support data-driven business decisions.

The complete workflow includes:
* Cleaning and preprocessing raw data using Python.
* Storing and querying data using PostgreSQL.
* Visualizing insights through an interactive Power BI dashboard.
The goal is to understand customer behavior, purchasing trends, and revenue drivers.

Dataset

* Real-world structured customer shopping dataset
* Format: CSV
* Includes:

  * Customer demographics
  * Transaction details
  * Purchase dates, products, and revenue information
